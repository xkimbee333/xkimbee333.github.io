กิจกรรมที่เคยเข้าร่วม / เคยทำ

  • การแข่งขันหุ่นยนต์ระดับกลาง ณ งานศิลปหัตถกรรมครั้งที่ 71 พ.ศ.2566

    • ผมได้เข้าร่วมการแข่งขันหุ่นยนต์ระดับกลางในงานศิลปหัตถกรรมครั้งที่ 71 ซึ่งเป็นเวทีที่เปิดโอกาสให้ผมพัฒนาทักษะด้านการออกแบบและควบคุมหุ่นยนต์ในสถานการณ์จริง การแข่งขันครั้งนี้ช่วยเสริมสร้างความเข้าใจด้านกลไกและการทำงานเป็นทีมอย่างมีระบบ

  • การแข่งขันหุ่นยนต์ระดับผสม ณ งานศิลปหัตถกรรมครั้งที่ 72 พ.ศ.2567

    • ในปีถัดมา ผมได้เข้าร่วมการแข่งขันหุ่นยนต์ระดับผสมในงานศิลปหัตถกรรมครั้งที่ 72 ซึ่งมีความซับซ้อนมากขึ้นทั้งในด้านการออกแบบ การเขียนโปรแกรม และการแก้ปัญหาเฉพาะหน้า เป็นประสบการณ์ที่ท้าทายและช่วยยกระดับทักษะด้านเทคโนโลยีของผมให้ก้าวไปอีกขั้น

  • อบรมและแข่งขันหุ่นยนต์ ณ โรงเรียนโล๊ะป่าตุ้ม พ.ศ.2567

    • ผมมีโอกาสเข้าร่วมการอบรมและแข่งขันหุ่นยนต์ที่เน้นการใช้การต่อบล็อกแทนการเขียนโค้ดแบบดั้งเดิม เป็นการเรียนรู้ที่เปิดมุมมองใหม่ในการเขียนโปรแกรมเชิงภาพ (Visual Programming) ซึ่งเหมาะสำหรับการพัฒนาแนวคิดเชิงตรรกะและการออกแบบระบบควบคุมหุ่นยนต์อย่างสร้างสรรค์

  • การฝึกสร้างเว็บไซต์และการออกแบบ UI ด้วย HTML CSS

    • การฝึกสร้างเว็บไซต์และการออกแบบ UI เว็ปไซต์ โดยการสร้างเว็ปไซต์ซื้อสินค้าออนไลน์โดยมีระบบเข้าสู่ระบบเพื่อรับโค้ดส่วนลดฟรี 6 หลักโดยโค้ดส่วนลดแต่ละรอบจะไม่เหมือนกันเพราะเป็นระบบสุ่มตัวอักษร มีหน้าแนะนำสินค้าและมีหมวดหมู่สินค้าแต่ละหมวดหมู่ให้เลือก แต่ละสินค้าจะมีรูปภาพประกอบและคำอธิบายสินค้าราคาบ่งบอกและมีตะกร้าสินค้าให้กดเลือกซื้อ มีการแจ้งเตือนเวลานำสินค้าเข้าตะกร้า


    คลิกเข้าเว็บไซต์
  • สร้างเว็บไซต์ PORTFOLIO ในการเก็บผลงาน

    • การสร้างเว็บไซต์ portfolio เพื่อใช้ในการเก็บผลงานหรือให้คนอื่นๆเข้าชมได้ง่ายขึ้น ทำให้การดูผลงานหรือการดูประวัติต่างๆทำได้ง่ายขึ้นเพียงแค่สแกนหรือค้นหาเว็บไซต์ก็สามารถดูผลงานหรือข้อมูลต่างๆ ได้อย่างสะดวกและง่ายดายกว่าการที่ต้องมานั่งเปิดไฟล์ PDF

  • การฝึกสร้างโปรแกรมตู้ ATM จำลองผ่าน Python

    • การฝึกสร้างตู้ ATM จำลองผ่านภาษา Python ซึ่งเป็นการเปิดประสบการณ์ใหม่ๆในการเขียนโปรแกรมเพราะเริ่มมีกระบวนการ การสร้างที่ซับซ้อนกว่า Html Css พอสมควร จึงต้องหาความรู้และทดลองสร้างจึงได้โปรแกรมนี้ขึ้นมา โปรแกรมตู้ ATM นี้มีเมนูอยู่ทั้งหมด 4 เมนู เช็คยอดเงิน ฝากเงิน ถอนเงิน ออกจากระบบ ถ้ากดเช็คยอดเงินจะทำการแสดงยอดเงินคงเหลือขึ้นมา ถ้าต้องการ ฝาก/ถอน จะมีหน้าให้ระบุจำนวนเงิน ฝาก/ถอน และรหัสยืนยัน ถ้าเกิดไม่ระบุรหัสยืนยันจะไม่สามารถทำธุรกรรมได้ ส่วนเมนูออกจากระบบคือออกจากโปรแกรม

  • การสร้างโปรแกรมเครื่องคิดเลขด้วยภาษา Python

    • การสร้างโปรแกรมเครื่องคิดเลขด้วยภาษา Python การสร้างโปรแกรมเครื่องคิดเลขนั้นเป็นการฝึกเขียนภาษาโปรแกรมมิ่งได้ดี เพราะได้ทั้งการเขียนโค้ดและการฝึกออกแบบ GUI ผมจึงเลือกสร้างโปรแกรมเครื่องคิดเลขขึ้นมา

  • การสร้างโปรแกรมเช็คพนักงานมาสายด้วยภาษา Python

    • การสร้างโปรแกรมเช็คพนักงานมาสายด้วยภาษา Python เป็นการฝึกเขียนภาษาและการเขียน Gui โดยมีพื้นฐานการเขียน Gui มาจากการสร้างเครื่องคิดเลข นำมาปรับใช้ในการเช็คพนักงานมาสายให้โปรแกรมมีหน้าตาสวยงามขึ้นและใช้งานง่าย เพียงแค่ใส่รหัสประจำตัวของพนักงาน และหมายเลข Pin ก็สามารถดูได้แล้วว่าพนักงานคนนั้นมาสายหรือไม่สายโดยการตั้งเวลาไว้ในโค้ด
